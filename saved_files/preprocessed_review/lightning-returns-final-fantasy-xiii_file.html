    As ports go, it's barely a token effort. If you've ever used a console emulator on a keyboard, you'll understand the experience.

    So far, I've had the game window open for two hours. One hour of that was working round two game breaking bugs just to get past the first cutscene. The rest of that time was trying to find controls that were in any way usable in order to get through the first battle.

    There has been no effort put in to make the game playable on a PC. There is no mouse support, and despite mapping keys all over the keyboard and separating out menu directional controls from movement directional controls (I really wish I were kidding), they've bound jump, cancel, and one of the attack buttons to backspace, another attack button to comma, and the last one to Enter.

    So you will have to use a gamepad to play this game. But what if the gamepad isn't the gamepad they used? Well you're still out of luck, as you can't remap gamepad buttons. But "Ahah!" you might say, "I can simply remap my joypad!". "No" the game replies, as every time you select a menu item, it stops taking any inputs until you click outside of the game window and click back inside it.

    While this game can be played up to 1080p / 60fps, the graphical options are a random mishmash of options featuring just about everything you never wanted, and nothing you did. Bizarrely, there isn't an anti-aliasing option in there, yet it does have the ominously labelled "Color correction" and "Glare adjustment".

    This port can be summed up by the message displayed after hitting new game which reads "Do not turn off your Playstation 3 system..."
    So if you're thinking about buying this game, consider that advice: Leave the PS3 on, and this poor excuse for a port off.    … Expand
