The graphics (even though the game is 10+ years old) still look as charming and as beautiful as they did back in 2000. The audio is great as you listen to the music inspired by ancient Egypt whilst hearing the sound effects of your bustling city. The game play is fun, varied and you will not get bored as there is always something to be doing. If you want to run this game on modern Windows, I've found it runs best on Windows XP SP1 - though this varies between people. If you get bored of the 100+ pre-built levels, you can create one using the in-game Map Editor which has a nice clean, easy to use interface which allowed me to create the maps I wanted with ease. The only criticism I have of the game is that it can get repetitive in the 'campaign' - as you have to keep on building cities with no real variation of objective. However, if you enjoy city builders then you will, without a doubt, love this game like I do. This is very close to a masterpiece, 9/10.… Expand
