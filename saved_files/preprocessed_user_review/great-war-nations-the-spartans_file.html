Gameplay tends to be slow, with combat changing the player's fortune rather quickly. Players can design their own units using a palette of weapons and armor collected and/or researched, but the collection of the components, the number of them, and the relative lack of healing units ensure a fairly tedious cycle of build -> combat -> build again. Heroes are merely super-powered infantry without much in the way of strategic abilities nor even token personalities, which was a disappointment for me.

    All in all, it's an 'OK' budget game by a budget studio, but don't expect the few novel ideas in the game to carry the day.
