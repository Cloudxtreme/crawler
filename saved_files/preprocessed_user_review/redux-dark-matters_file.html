The music in this game is just phenomenally good. And the game is very responsive to controls.

    Most levels have been designed well and the game is really fun to play. Even for the veterans of the genre. However there are certain spots in some levels which require what I'd call "trained eye" to pass. It isn't a fundamental fault in the game but some people might think that it's a fault in the level design. There's a slight learning curve associated with the game.

    The game lacks proper integration with the Steam. There are no achievements, no Steam cards, and no support for SteamOS/Linux and/or OS X. Also the graphics settings menu uses slightly obscure English which makes it bit difficult to adjust the graphics to look as good as possible and requires bit of trial and error. There is also no global Steam scoreboard which means you cannot compete with your friends in ladders.

    All in all REDUX is a really fun shoot em' up. It doesn't offer anything new to the genre but rather plays it safe by making sure that all the basic fundamental things are executed as well as possible.
