Summary: RollerCoaster Tycoon 3 takes the series to new heights with brand new features including 3-D graphics, giving players complete control of views around the park, and a front seat to the action with a new Coaster Cam feature that allows players to ride the roller coasters. RollerCoaster Tycoon 3 also features new gameplay elements that fans have been asking for - including diversity of guests, added group behavior and improved guest AI - and many other new features which build upon the fun and addictive nature of the original game.
