Summary: A 3D real-time strategy game set in the 1930's that pits the player against an evil madman in a desperate campaign across a chain of remote and diverse islands. Using Earth's most formidable animals as building blocks, the player creates an army of unique, mutant monsters in a struggle to protect an unsuspecting world. [Microsoft]
