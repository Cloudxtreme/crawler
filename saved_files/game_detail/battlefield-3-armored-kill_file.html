Summary: Battlefield 3: Armored Kill ups the ante for vehicular mayhem as only Battlefield can do. Featuring new drivable tanks, ATVs, mobile artillery and more, this expansion pack also delivers huge battlefields for an all-out vehicle assault. Armored Kill also includes the biggest map in Battlefield history. KEY FEATURES: * All-out vehicle warfare on four new maps * Includes the biggest map in Battlefield history * Drive five new vehicles, including tank destroyers, mobile artillery and an ATV or spawn on the gunship and man the cannons * Earn 20+ unlocks for the new vehicles * New Tank Superiority game mode * Earn five new Assignments
