Summary: The game is set in year 2010 where player's conscience is entrapped by an up-to-date virus-infected virtual reality-playing machine called FireStarter. The virus-caused failure resulted in computer changing the rules of the game - to break out of the trap, the player is to complete the game within 48 hours. Immerse in an unbelievable world, created by super-modern playing machine, where you are to make it a winner of the adrenaline-brimming action. [GSC Game World]
